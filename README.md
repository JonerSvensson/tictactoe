# Tic-Tac-Toe 

## Features
- **Two-player gameplay**  
  Two players take turn picking a cell to put their mark on.

- **Play against bot**  
  Option to choose if the opponent (player2) should be a bot or not. The bot picks random cells when playing.

- **Win and draw detection**  
  The game automatically checks after every move whether a player has won or if the game ends in a draw.

- **Move validation**  
  Each turn, the game verifies that the playerâ€™s input refers to a valid and empty cell.

- **Automatic restart after a win**  
  When a player wins the board resets and a new game begins without restarting the program. 

- **Player name input**  
  At the start, players enter their names. The game displays whose turn it is each round.

- **Score tracking**  
  The game keeps track of how many rounds each player has won.

- **Input handling**  
  All player inputs are handled so the program does not crash due to incorrect input.
